ESP8266 Home Automation — Relay + DHT22

A Blynk-connected ESP8266 home automation system with a relay module for remote appliance switching, DHT22 temperature and humidity sensing, and an SH1106 OLED display for local readout. Controlled via the Blynk mobile app with virtual pins.

Project 07
Arduino / IoT

Relay Control Module

A simple but foundational Arduino relay control sketch using a NodeMCU/ESP board. Demonstrates active-LOW relay toggling with precise timing — the building block used across multiple home automation projects.

// Active-LOW Relay Control
int relayPin = D5;
digitalWrite(relayPin, LOW);
// ON — wait 2s
digitalWrite(relayPin, HIGH);
// OFF — wait 2s

Thesis

Academic capstone research.

BSIT Capstone · TUP Manila · 2026

A Real-Time Multi-Sensor Detection Using ESP32
for Home Security and Monitoring System

A capstone research project developed with a team of four co-researchers. The system uses an ESP32-S3 with dual primary sensors — MQ-2 for gas/smoke detection and IR flame sensor for fire detection — plus a DHT22 for ambient temperature context. A Flutter mobile app connects via Supabase for real-time push alerts, automated SMS queueing, and PWM servo-based actuator control. A camera with pan-tilt servo is included for live monitoring.
